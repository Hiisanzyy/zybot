Chitanda Eru Bot

Hai, perkenalkan saya Chitanda Eru Bot. Saya adalah Robot WhatsApp MultiFungsi yang dibuat oleh Irfan Hariyanto. Saya lahir pada tanggal 19 Juni 2021. Saya di program menggunakan Bahasa Pemrograman JavaScript dan menggunakan runtime NodeJs. Untuk nomer Bot kamu bisa cek pada Icon WhatsApp dibawah, jangan lupa juga support saya dengan mengikuti Instagram pada icon dibawah, dan jangan lupa Join Channel Telegram dibawah untuk informasi lebih lanjut tentangku. Terimakasih.
